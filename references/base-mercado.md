# Base de Mercado Viva

Arquivo **vivo**: acumula os achados de pesquisa de mercado entre casos, para **calibrar** sizing,
preço e concorrência sem re-pesquisar do zero e manter **consistência**. Toda pesquisa da Fase 3
alimenta este arquivo. **Regra:** todo número é **datado** e tem **confiança**; ao usar um dado,
cheque a data — se estiver velho (> ~6 meses para dados voláteis como Selic/hype), **revalide**.

> Como usar: antes de pesquisar do zero, consulte aqui. Depois da pesquisa, **adicione/atualize** as
> entradas relevantes (com data e fonte). Não apague histórico útil — marque como desatualizado.

## Whitespaces identificados
- **"Preserve seu julgamento na era da IA" / soberania cognitiva** para C-level — demanda alta, oferta
  quase toda no ângulo oposto ("use mais IA"). Categoria imatura (risco), mas aberta. _(jul/2026)_
- **Cohort ao vivo puro** é raro no BR (dominam assinatura on-demand e imersão presencial). _(jul/2026)_
